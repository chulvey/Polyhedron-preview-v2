Polyhedron Preconstructed – Data-Driven Prototype (V5)

FILES:
- index.html
- styles.css
- app.js
- decks.json
- /decks/*.html (placeholder pages)

UPLOAD TO GITHUB PAGES:
1) Upload index.html, styles.css, app.js, decks.json to the repo root.
2) Upload the 'decks' folder.
3) (Optional) Add chibi images into: assets/chibis/
   - If missing, a built-in placeholder will be shown (no broken icons).

UNLOCK LOGIC:
- Unlocked = first 4 decks in Set 1 Mono White (from Index sheet)

TIER LOGIC:
- Wave 1 -> Beginner
- Wave 2 -> Advanced
- Wave 2.5 -> Expert

ART COMPLIANCE:
- Locked: never shows card art
- Unlocked: press-and-hold (mobile) or hover (desktop) shows Scryfall image_uris.small
- Label appears only while art is shown
- Turn off instantly by setting ART_ENABLED=false in index.html
